**Liquid Glass SVG Recipe (apply to every icon):**

Each icon's SVG should include a `<defs>` block with:

1. **A linear gradient** (`id="{name}Grad"`) going from a lighter, semi-transparent version of the accent color (top, ~40% opacity) to a slightly darker version (bottom, ~15% opacity). This creates the translucent glass fill.

2. **A highlight gradient** (`id="{name}Highlight"`) — white going from ~25% opacity at top to 0% at bottom. This creates the frosted inner glow / specular highlight.

Then the icon's main shape(s) should have:
- **Filled shape** using the glass gradient (`fill="url(#{name}Grad)"`) — gives the translucent colored glass body
- **Stroke** using the full accent color at ~60-70% opacity (`stroke` with `strokeOpacity`) — subtle border definition
- **Highlight overlay** — a slightly smaller/inset copy of the main shape filled with the white highlight gradient, or a small rect/ellipse at the top of the shape

**Key design principles:**
- Keep each icon's recognizable silhouette — same paths/shapes, just add glass treatment
- The primary shape gets the gradient fill + subtle stroke
- Secondary/detail shapes stay as lighter accent strokes (existing opacity pattern) but can get a very faint fill too
- Don't over-complicate — 2-3 layers per icon max (glass fill + highlight + detail)
- No filter effects (blur, feGaussianBlur) — these are expensive at small sizes. Use gradients only.
- Keep the same accent color per icon — just add glass depth to it
- The `fill="none"` stroke-only approach is replaced with filled shapes that have translucent gradients

**Example transformation for DevicesIcon:**

Before (flat):
```jsx
<rect x="2" y="3" width="13" height="10" rx="2" stroke="#4A9EFF" strokeWidth="1.5" fill="none" />
```

After (liquid glass):
```jsx
<defs>
  <linearGradient id="devGrad" x1="0" y1="0" x2="0" y2="1">
    <stop offset="0%" stopColor="#4A9EFF" stopOpacity="0.4" />
    <stop offset="100%" stopColor="#4A9EFF" stopOpacity="0.12" />
  </linearGradient>
  <linearGradient id="devHighlight" x1="0" y1="0" x2="0" y2="1">
    <stop offset="0%" stopColor="#ffffff" stopOpacity="0.25" />
    <stop offset="100%" stopColor="#ffffff" stopOpacity="0" />
  </linearGradient>
</defs>
<rect x="2" y="3" width="13" height="10" rx="2" fill="url(#devGrad)" stroke="#4A9EFF" strokeWidth="1" strokeOpacity="0.6" />
<rect x="2.5" y="3.5" width="12" height="5" rx="1.5" fill="url(#devHighlight)" />
```

Apply this pattern to all 15 icons. Preserve the shape/concept of each icon but make them feel like frosted glass pieces with depth and glow.

**DO NOT change:**
- The `IconProps` interface
- The `moduleIconMap` export
- The `moduleLabelMap` export
- The component function names
- The accent color assigned to each icon

**Reduce strokeWidth** from 1.5 to 1 on glass-filled shapes (the fill provides visual weight, heavy strokes look wrong on filled shapes).
