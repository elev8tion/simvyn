  <action>
    **PlatformAdapter extension (device.ts):**
    Add optional method: `openUrl?(deviceId: string, url: string): Promise<void>`

    **iOS adapter (ios.ts):**
    Implement openUrl: `execFileAsync("xcrun", ["simctl", "openurl", deviceId, url])`
    This handles http/https URLs, custom schemes (myapp://), and universal links.

    **Android adapter (android.ts):**
    Implement openUrl: `execFileAsync("adb", ["-s", deviceId, "shell", "am", "start", "-a", "android.intent.action.VIEW", "-d", url])`
    Guard against `avd:` prefix device IDs. This handles http/https, custom schemes, and intent URIs.

    **Create `packages/modules/deep-links/` module:**

    **package.json**: name `@simvyn/module-deep-links`, type module, dependencies on `@simvyn/types`, `@simvyn/core`, `@simvyn/server`.

    **tsconfig.json**: Same pattern as other modules.

    **routes.ts** — Fastify routes (prefix `/api/modules/deep-links`):
    - `POST /open` — Body: `{ deviceId, url }`. Look up device, get adapter, call adapter.openUrl(). Return `{ success: true, url, deviceId }`. Save to recent history.
    - `GET /favorites` — Return all saved favorites from storage. Each favorite: `{ id: string, url: string, label: string, bundleId?: string, createdAt: string }`.
    - `POST /favorites` — Body: `{ url, label, bundleId? }`. Add to favorites with generated UUID id. Return the created favorite.
    - `DELETE /favorites/:id` — Remove favorite by id. Return `{ success: true }`.
    - `GET /history` — Return recent deep link launches (last 50). Each: `{ url, deviceId, deviceName, timestamp }`.

    Use `createModuleStorage("deep-links")` for persistence (keys: "favorites", "history").

    **manifest.ts** — Module manifest:
    - name: "deep-links", icon: "external-link", capabilities: ["deepLinks"]
    - register: register routes (no WS handler needed — deep link opening is fire-and-forget)
    - cli: `simvyn link <device> <url>` — opens URL on device. Same CLI adapter pattern: create own adapters + DeviceManager, look up device by ID/prefix, call adapter.openUrl(). Print confirmation to stdout. Cleanup on exit.
  </action>
