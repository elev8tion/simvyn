## Decisions Made
- Used tsx loader (`--import tsx`) instead of `--experimental-strip-types` because mock.module's loader hook conflicts with the type-stripping loader (both register ESM loader hooks, mock loader can't resolve `.js` → `.ts`)
- Used `any` types for mock ChildProcess/stdin/spawn returns to avoid strict TypeScript narrowing issues in test mocks (types stripped at runtime anyway)
- Skipped IPA install path test since it requires filesystem mocking (mkdtemp/unzip/readdir) — the .app path validates the core simctl invocation
- collectBugReport test wrapped in try/catch since fs.stat fails on non-existent file (mocked exec doesn't create actual files) — command assertion still verifies correct invocation
