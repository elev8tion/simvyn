- Cellular bars select → glass-select
- WiFi bars select → glass-select
- Network operator input → glass-input
- "Apply" button → glass-button-primary
- "Clear Overrides" button → glass-button-destructive
- Each control row — use consistent spacing: flex items-center justify-between gap-3 with label on left and control on right

IMPORTANT: Some controls may be conditionally hidden based on device capabilities. Keep all conditional rendering logic. Only replace visual classes.
