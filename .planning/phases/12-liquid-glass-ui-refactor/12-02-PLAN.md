**CSS changes in main.css — update .dock-sidebar:**

Transform the sidebar from a full-height flush panel into a floating pill-shaped dock:

```css
.dock-sidebar {
  position: relative;
  z-index: 20;
  display: flex;
  flex-direction: column;
  align-items: center;
  padding: 8px 0;
  gap: 2px;
  width: 52px;
  margin: 12px 0 12px 12px;      /* Float away from edges */
  border-radius: 16px;            /* Pill shape — rounded all corners */
  backdrop-filter: blur(24px) saturate(1.5);
  -webkit-backdrop-filter: blur(24px) saturate(1.5);
  background: rgba(40, 40, 55, 0.45);  /* More translucent than before */
  border: 1px solid rgba(255, 255, 255, 0.10);
  box-shadow:
    0 8px 32px rgba(0, 0, 0, 0.3),
    inset 0 0.5px 0 rgba(255, 255, 255, 0.08);
  overflow-y: auto;
  overflow-x: hidden;
  align-self: center;              /* Center vertically in flex parent */
}
```

Key changes from old dock:
- Width: 60px → 52px (tighter)
- Added margin (12px left, 12px top/bottom) to float away from edges
- border-right removed → full border: 1px solid on all sides
- Added border-radius: 16px (was 0/none — flush against edge)
- Added box-shadow with inset highlight (dock itself is now a glass element)
- background more translucent: rgba(40, 40, 55, 0.45) — was rgba(16, 16, 24, 0.7)
- Stronger blur: 24px (was 20px)
- align-self: center to vertically center in the flex container
- padding: 8px 0 (was 12px 0) — tighter for pill shape
- gap: 2px (was 4px) — tighter spacing
