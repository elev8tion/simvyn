  <action>
Run grep-based audits across all 12 non-location panel files and fix inconsistencies:

**1. Padding consistency:**
`grep -rn "p-4\b\|p-5\b" packages/dashboard/src/panels/ --include="*.tsx"` — find panels not using p-6.
Standardize: all panel root containers should use `p-6` (24px). Update any p-4 or p-5 to p-6.

**2. Panel header pattern:**
Check each panel's `<h1>` element: should be `text-base font-medium`. Check header container: should be `flex items-center justify-between mb-4`.
Fix any that deviate (e.g., text-lg, mb-3, missing justify-between).

**3. Border-radius audit:**
`grep -rn "rounded-xl\|rounded-2xl\|rounded-lg\|rounded-md" packages/dashboard/src/panels/ --include="*.tsx"` — find hardcoded radius values.
Containers → `rounded-[var(--radius-panel)]` or inherit from glass-panel.
Controls → `rounded-[var(--radius-button)]` or inherit from glass-button/glass-input/glass-select.
Keep Tailwind radius classes only where they're semantically different (e.g., rounded-full for dots/avatars).

**4. Glass-on-glass nesting audit:**
`grep -rn "glass-panel" packages/dashboard/src/panels/ --include="*.tsx"` — verify no component nests glass-panel inside glass-panel.
Common offenders: Database inner components, Settings sections, CrashLogs filter panel.
Fix: inner elements → `bg-bg-surface/10` or `bg-transparent`, not glass-panel.

**5. Empty state audit:**
`grep -rn "text-center.*text-text-secondary\|No.*selected\|No.*found" packages/dashboard/src/panels/ --include="*.tsx"` — verify all empty states use glass-empty-state class.
  </action>

  </verify>
