Create `packages/core/src/favourites.ts` using `createModuleStorage("favourites")` from `./storage.ts`. Store data at `~/.simvyn/favourites/favourites.json` as a flat `string[]` of device IDs.

Export these functions:
- `getFavourites(): Promise<string[]>` — read the array, return `[]` if missing
- `addFavourite(id: string): Promise<void>` — add ID if not already present, write back
- `removeFavourite(id: string): Promise<void>` — remove ID if present, write back
- `cleanupStaleFavourites(currentDeviceIds: string[], missCountMap: Map<string, number>, threshold?: number): Promise<string[]>` — for each favourite ID not in `currentDeviceIds`, increment its count in `missCountMap`. If count exceeds `threshold` (default 3), remove from favourites. Reset count for IDs that reappear. Return the cleaned list. This handles temporarily disconnected USB devices by only removing after multiple consecutive absences.

Also re-export from `packages/core/src/index.ts` (add to existing exports).

Order favourites alphabetically within the stored array per Claude's discretion for consistent ordering.
