Create `packages/types/src/collections.ts` with these type definitions:

```typescript
// Action parameter types
ActionParamType = "string" | "number" | "boolean" | "select"

ActionParam = {
  key: string;
  label: string;
  type: ActionParamType;
  required: boolean;
  options?: { label: string; value: string }[];  // for "select" type
  placeholder?: string;
  defaultValue?: unknown;
}

ActionDescriptor = {
  id: string;
  label: string;
  description: string;
  module: string;               // grouping label (e.g. "device-settings", "location")
  params: ActionParam[];
  execute: (adapter: PlatformAdapter, deviceId: string, params: Record<string, unknown>) => Promise<void>;
  isSupported: (adapter: PlatformAdapter) => boolean;
}

CollectionStep = {
  id: string;                   // UUID
  actionId: string;             // references ActionDescriptor.id
  params: Record<string, unknown>;
  label?: string;               // optional user-friendly override
}

Collection = {
  id: string;                   // UUID
  name: string;
  description?: string;
  steps: CollectionStep[];
  schemaVersion: 1;
  createdAt: string;            // ISO 8601
  updatedAt: string;            // ISO 8601
}
```

Import `PlatformAdapter` from `./device.js` (type-only import).

Then update `packages/types/src/index.ts` to re-export all new types:
```typescript
export type { ActionDescriptor, ActionParam, ActionParamType, Collection, CollectionStep } from "./collections.js";
```

**Create `packages/modules/collections/action-registry.ts`:**

Export `actionRegistry` as an array of `ActionDescriptor` objects and a `getActionDescriptors()` function that returns the full array.

Register all 14 actions from the research. Each descriptor has:
- `id`, `label`, `description`, `module` (grouping)
- `params`: typed ActionParam[] with correct type/options/required
- `execute`: calls the corresponding PlatformAdapter method (e.g. `adapter.setAppearance!(deviceId, params.mode as "light" | "dark")`)
- `isSupported`: checks `!!adapter.methodName` (e.g. `(adapter) => !!adapter.setAppearance`)

The 14 actions:

| id | label | module group | adapter method | params |
|----|-------|-------------|----------------|--------|
| set-appearance | Set Appearance | device-settings | setAppearance | mode: select ["light","dark"] |
| set-location | Set Location | location | setLocation | lat: number, lon: number |
| clear-location | Clear Location | location | clearLocation | (none) |
| set-clipboard | Set Clipboard | clipboard | setClipboard | text: string |
| set-locale | Set Locale | device-settings | setLocale | locale: string |
| set-status-bar | Set Status Bar | device-settings | setStatusBar | overrides: string (JSON) |
| clear-status-bar | Clear Status Bar | device-settings | clearStatusBar | (none) |
| grant-permission | Grant Permission | device-settings | grantPermission | bundleId: string, permission: string |
| open-deep-link | Open Deep Link | deep-links | openUrl | url: string |
| add-media | Add Media | media | addMedia | filePath: string |
| launch-app | Launch App | app-management | launchApp | bundleId: string |
| terminate-app | Terminate App | app-management | terminateApp | bundleId: string |
| set-content-size | Set Content Size | device-settings | setContentSize | size: select ["xSmall","small","medium","large","xLarge","xxLarge","xxxLarge"] |
| set-increase-contrast | Set Increase Contrast | device-settings | setIncreaseContrast | enabled: boolean |

For the `set-status-bar` action, the overrides param should be type "string" with placeholder "JSON object" — the execution function parses it with `JSON.parse`.
