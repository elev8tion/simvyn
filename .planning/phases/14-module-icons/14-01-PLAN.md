<task type="auto">
  <name>Task 1: Create custom liquid glass SVG icons for all 13 modules</name>
  <files>packages/dashboard/src/components/icons/module-icons.tsx</files>
  <action>
Create `packages/dashboard/src/components/icons/module-icons.tsx` containing:

1. **13 SVG icon components** as React functional components, each accepting `{ size?: number, className?: string }` props. Each icon should be a hand-crafted inline SVG (NOT an image import, NOT Lucide) at a 24x24 viewBox.

2. Each icon should use a **unique accent color** with a Liquid Glass feel:
   - `devices` — Blue (#4A9EFF) — monitor/phone outline with device silhouette
   - `location` — Coral/Orange (#FF6B6B) — map pin with subtle glow
   - `apps` — Green (#4ADE80) — grid of 4 rounded squares (app grid)
   - `logs` — Amber/Yellow (#FBBF24) — scrolling terminal/text lines
   - `screenshot` — Violet (#A78BFA) — camera viewfinder frame
   - `deep-links` — Cyan (#22D3EE) — chain link or external arrow
   - `push` — Pink (#F472B6) — notification bell with dot
   - `fs` — Teal (#2DD4BF) — folder with file peek
   - `database` — Indigo (#818CF8) — stacked cylinder/discs
   - `settings` — Slate/Gray (#94A3B8) — gear with inner circle
   - `crash-logs` — Red (#F87171) — warning triangle with exclamation
   - `media` — Emerald (#34D399) — image/photo frame with mountains
   - `clipboard` — Sky (#38BDF8) — clipboard board with checkmark

3. Each SVG should have these Liquid Glass characteristics:
   - Rounded strokes (strokeLinecap="round", strokeLinejoin="round")
   - Stroke width of 1.5-2 (consistent across all icons)
   - The accent color used for the PRIMARY strokes/fills
   - A subtle secondary element in lower opacity (e.g., opacity="0.3") for depth
   - Clean, recognizable silhouette at 20x20 display size (dock icon size)

4. Export a **`moduleIconMap`** record mapping module name → component:
   ```tsx
   export const moduleIconMap: Record<string, ComponentType<IconProps>> = {
     devices: DevicesIcon,
     location: LocationIcon,
     // ...
   };
   ```

5. Export a **`moduleLabelMap`** record mapping module name → display label (moved from Sidebar.tsx):
   ```tsx
   export const moduleLabelMap: Record<string, string> = {
     devices: "Devices",
     location: "Location",
     // ...
   };
   ```

Design guidelines:
- Icons should feel like iOS/macOS system icons — clean, geometric, slightly rounded
- Each icon's accent color should be clearly visible at 20x20 in the dark sidebar
- Don't over-detail — 2-3 strokes max per icon for clarity at small sizes
- Use currentColor for any secondary/neutral strokes so they adapt to theme
