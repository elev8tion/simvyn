<task type="auto">
  <name>Task 1: Add active-only filtering with toggle to DeviceSelector</name>
  <files>packages/dashboard/src/components/DeviceSelector.tsx</files>
  <action>
Add local component state `showAll` (boolean, default `false`) to DeviceSelector. Reset `showAll` to `false` every time the dropdown opens (in the button onClick handler, when transitioning from closed to open).

Filter the `devices` array before passing to `groupDevices`:
- When `showAll` is false: filter to only `d.state === "booted"` devices
- When `showAll` is true: pass all devices through (no filter)
- IMPORTANT: Always include currently selected devices in the filtered list regardless of state — prevents the label showing "No devices" when a shutdown device was previously selected
- No special casing for favourites — they follow the same filter

Add a toggle row at the bottom of the dropdown (below all device groups, above the "No devices detected" empty state). Design:
- Full-width row with horizontal layout: label text on left, toggle indicator on right
- Label: "Show inactive devices" (when showAll is false) / "Showing all devices" (when showAll is true)
- Use a small inline toggle/switch indicator: a 32x18px pill-shaped container (rounded-full) with a sliding circle inside. Background: `bg-text-muted/20` when off, `bg-accent-blue/40` when on. Circle: `bg-text-muted/60` when off, `bg-accent-blue` when on. Transition the circle position with `translate-x`.
- The entire row is clickable (not just the toggle indicator)
- Row styling: `px-3 py-2 text-xs text-text-muted hover:bg-[rgba(255,255,255,0.05)] cursor-pointer` with a subtle `border-t border-white/5` separator above it
- Row should use a `<button>` element with `type="button"`

When `showAll` is false and there are shutdown devices that are being hidden, show a count hint in the toggle label: "Show N inactive devices". When all devices are already booted, still show the toggle but with "Show inactive devices" (no count). Derive count from `devices.length - filteredDevices.length`.

Update the empty state: when `devices.length > 0` but all are filtered out (no booted devices), show "No active devices" with a smaller "Toggle below to show inactive devices" hint instead of "No devices detected". Keep "No devices detected" for the true zero-devices case.
